Top 50 Busiest Airports in Europe 2017

The top 50 Airports in Europe, 2017 ranked by total passengers per year, including both terminal and transit passengers. Hover over a bubble to read the name of the airport, the city served and the number of passengers for 2017 per airport. The circle size is based on the number of passengers for the year 2017. Darker red circles represent airports with a greater increase rate.

Changes 2016-2017

This data correlates the airport rank to the passenger increase rate for the top 50 airports in Europe for the year 2017. This chart plots airport rank along the X axis and the passenger increase or decrease rate along the Y axis. The color and size of each bubble represent the number of passengers in 2017.

Changes 2015-2016

This data correlates the airport rank to the passenger increase rate for the top 50 airports in Europe for the year 2016. This chart plots airport rank along the X axis and the passenger increase or decrease rate along the Y axis. The color and size of each bubble represent the number of passengers in 2016.

Number of Passengers Carried in 2017

Each bubble represents an airport, color and size represent the number of passengers in 2017.
Hover over a bubble to read the name of the airport, the city served and last year's number of passengers.
